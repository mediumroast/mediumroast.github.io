Open source projects

Mediumroast, Inc. is excited to announce its contribution to the open-source community with three innovative projects: mediumroast_js, mediumroast_py, and company_dns. These projects have been designed to enhance software development, not only within our own mediumroast.io application but also more generally in the case of company_dns. By making these tools freely available, we aim to empower developers of all skill levels to build high-quality software more efficiently. At Mediumroast, we recognize the vital role of open-source software in the growth and evolution of the industry, and we are proud to play a part in this collaborative effort.

mediumroast_js

The Node.js JavaScript SDK for mediumroast.io provides a wrapper for the backend's RESTful APIs, a high-level interface for generating Microsoft Word reports, and core and helper command line utilities for managing companies, studies, interactions, and user objects. The SDK is designed to offer a first-class development experience and is regularly updated for reliable use by developers.

License: Apache-2.0 license

mediumroast_py

The Python SDK for mediumroast.io simplifies the development of ETL, machine learning, and natural language processing tools for the platform. Its powerful features, ease of use, and reliability make it an invaluable tool for developers. With a well-documented API and examples in the mediumroast_examples/ directory, the SDK is suitable for both novice and experienced developers.

License: Apache-2.0 license

company_dns

company_dns is a software package that uses data from the SEC EDGAR repository and Wikipedia to make it faster and more efficient to gather information about companies. This tool automates the information gathering process, providing a more comprehensive view of companies and their activities. company_dns has many potential applications, such as investment analysis, competitive intelligence, or academic research.

License: Apache-2.0 license
