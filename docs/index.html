Open source projects

Mediumroast, Inc. is excited to contribute to the open-source community with four projects: mediumroast_js, mediumroast_py, company_dns, and mr_markdown_builder. These projects have been designed to enhance software development, not only within our own mediumroast.io application but also more generally in the case of company_dns and mr_markdown_builder. 

mediumroast_js

Mediumroast for GitHub Node.js based CLI and API/SDK. Mediumroast for GitHub intends to help Products oranizations construct an active interactions repository close to the action of development and issue management in GitHub. 

License: Apache-2.0 license

mediumroast_py

Mediumroast for GitHub Python based API/SDK.

License: Apache-2.0 license

company_dns

company_dns, is an open source micro-service focused that provides company data from EDGAR plus Wikipedia, and SIC lookup.

License: Apache-2.0 license

mr_markdown_builder

A fork of the venerable yet archived markdown-builder with improvements including, but not limited to: Tables, Quotes, Task lists, Inline code blocks, Long code blocks, Badges from shields.io, GeoJson, TopoJson, and many more.

License: Apache-2.0 license
